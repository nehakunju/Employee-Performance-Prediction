# Employee Performance Prediction

## Overview

This project predicts employee performance using Machine Learning. It was developed using Python and Streamlit to analyze employee-related data and generate performance predictions.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Files

- app.py - Streamlit application
- MLPR.ipynb - Jupyter Notebook
- employees_final_dataset.csv - Dataset
- features.pkl - Feature information
- label_encoders.pkl - Label encoder objects
- scaler.pkl - Scaler object

## Project Workflow

1. Load dataset
2. Data preprocessing
3. Feature engineering
4. Train Machine Learning model
5. Predict employee performance
6. Display prediction using Streamlit
